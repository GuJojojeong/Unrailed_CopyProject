## 🤗 프로젝트 소개
🥏 Unrailed_CopyProject는 기차게임 Unrailed를 기반으로 만든 공부용 카피게임입니다!


 멀티플레이를 통해 다른 친구들과 함께 기차게임을 즐겨보세요! (근데 플레이가......)

 
## 🕹 조작방식
상하좌우 이동키 - 이동
스페이스바 - 잡기
도구를 든 상태로 나무나 돌에 가까이 다가가면 자동으로 자원을 캡니다!
